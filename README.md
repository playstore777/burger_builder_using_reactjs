## Run Locally

Clone the project

```bash
  git clone git@github.com:playstore777/burger_builder_using_reactjs.git
```

Go to the project directory

```bash
  cd burger_builder_using_reactjs
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
